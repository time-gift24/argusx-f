# Zard UI Button

来源: https://zardui.com/docs/components/button

## API

| Property | Type | Default |
|----------|------|---------|
| zDisabled | boolean | false |
| zFull | boolean | false |
| zLoading | boolean | false |
| zShape | default \| circle \| square | default |
| zSize | default \| xs \| sm \| lg \| icon \| icon-xs \| icon-sm \| icon-lg | default |
| zType | default \| destructive \| outline \| secondary \| ghost \| link | default |

## Selectors

- `z-button`
- `button[z-button]`
- `a[z-button]`

## Notes

- 使用 Angular Signals
- 支持 icon 检测 (MutationObserver)
- 无 CVA 导出
