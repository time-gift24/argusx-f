# URL Reference

## Standard Query Parameters

All preview URLs support these query parameters:

| Parameter | Description | Example Value |
|-----------|-------------|---------------|
| item | Component identifier | select-example |
| style | Visual style | mira, default, new-york |
| theme | Color theme | cyan, neutral, violet |
| font | Font family | nunito-sans, inter |
| menuAccent | Menu accent | bold, default |
| radius | Border radius | medium, small, large |
| template | Framework | vite, next |

## Fallback URL

If component preview returns 404, use the create page to find links:
```
https://ui.shadcn.com/create?base=radix&style=mira&baseColor=neutral&theme=cyan&iconLibrary=lucide&font=nunito-sans&menuAccent=bold&menuColor=default&radius=medium&template=vite&rtl=false
```

Extract component links from the left navbar navigation.
